# 果果的秘密空间 — 创意设计

> 创建时间：2026-06-10
> 状态：创意设计完成，待分期规划

## 项目概述

为果果打造一个名为"秘密空间"的个人网站。整体是一个卡通风格的温馨儿童房间——"果果的魔法小屋"，房间里的物品都是可交互的功能入口。面向熟人展示，设置密码校验。

## 视觉风格

- AI 生成底图 + 关键交互物品用 SVG
- 卡通、温馨、低幼可爱、色彩鲜明、大圆角
- 一屏固定视角，响应式适配移动端
- 全局自定义卡通光标

## 空间层级设计

采用**单房间 + 局部 Zoom-in** 方案：

```
门外 → 房间全景（区域入口 + 猫 + 果果形象）
        ├→ 书桌区域（zoom-in）
        ├→ 书架区域（zoom-in）
        ├→ 墙面区域（zoom-in）
        ├→ 窗户区域（zoom-in）
        ├→ 游戏区域（zoom-in）
        └→ 返回全景
```

- 全景作为导航枢纽，视觉留白、不拥挤
- 每个区域 zoom-in 后是独立的详细场景
- 过渡用镜头推进动画
- 扩展只需往对应区域的特写里加物品，全景无需改动

## 入口体验

### 密码解锁

- 房间门锁着，输入密码解锁
- 开门过场动画 + 吱呀声效
- 访客 vs 果果不同权限（访客只能看公开部分）

### 加载体验

- 等待画面：门慢慢打开 / 猫在门缝偷看
- 随机 loading 小贴士（"你知道吗？果果X岁时学会了骑车！"），数据池可持续添加

### 开场

- 第一次进入有简短开场文字/语音："欢迎来到果果的秘密空间，这里收藏着她所有的宝贝回忆……"

## 全景层元素

### 果果虚拟形象

- 站在房间里的小卡通人
- 随年龄更新形象
- ~~可换衣服/换发型~~（defer：美术资源量大，未来有精力再做）

### 两只猫 — CPU（英短蓝白）& 好甜（英短蓝猫）

**日常行为：**
- 白天走动/玩耍，晚上蜷缩睡觉

**互动：**
- 点击有反应：喵叫、翻身、蹭蹭

**捣乱：**
- 偶尔趴书架推书、跳书桌挡日记本，点击赶走

**长时间无操作：**
- 蹭镜头挡画面，点击走开

### 镜子

- 全景层可见
- 点击进入"镜像房间"（一切是反的，藏着只能从镜中世界拿到的东西）

### 积木

- 地板散落，可拖动
- 拼成简单图形后触发小动画奖励

### 氛围元素

- 天花板星星贴纸（夜间发光闪烁）
- 早安/晚安黑板（门上，根据访问时间显示）
- 足迹统计（角落小仪表盘："果果的房间被参观了 X 次"）
- 季节性房间变化（详见下方"四季系统"）
- 节日装饰（详见下方"节日系统"）
- 粒子特效（详见下方"氛围特效"）

## 墙面区域

### 冒险地图（中国地图）

- 静态有颜色区分，去过的省份被点亮
- hover 省份悬浮高亮
- 点击省份打开瀑布流照片
- 显示"已点亮 X/34 个省级行政区"
- 数据通过管理后台维护

### 全家福墙

- 一面照片墙，挂着全家福
- 点击某张进入那次的故事

### 身高尺

- 标记果果每年的身高
- 点击某个刻度弹出那年的一张照片

### 日历/翻页钟

- 显示真实日期时间
- 点击查看"今天的果果"：展示当天拍过的历史照片（"去年今天你在做什么"）
- 内含倒计时（下一个特殊日子）

### 成长贴纸墙

- 每解锁一个里程碑就多一张贴纸
- 里程碑示例：第一次涂鸦、第一篇日记、去过5个省……

### 变化日志

- 隐蔽角落
- 用童话语言写的更新日志（"果果的房间最近变化了什么"）

## 书架区域

### 年度相册（记忆之书）

- 每年一本，随年龄增长增加
- hover 相册抽出动画
- 点击打开，全屏翻页浏览（像真实相册）
- 每本相册完全独立设计（外壳 + 内页风格每年不同）
- 图片走对象存储

### 成长时间线

- 半隐藏入口（日历上一个不太明显的小图标，知道的人能找到）
- 点击进入竖向时间线
- 按年/月展示关键事件

---

## 书桌区域

### 日记本（带密码锁）

- 密码可修改
- 写日记 + 自由涂鸦画板
- 数据持久化到后端

### 音乐盒

- 点击播放音乐

### 星星罐

- 玻璃罐，每个重要时刻往罐里加一颗星星
- 点击星星回放那个时刻

### 每日一句

- 便利贴/小台历，每天展示一句给果果的话
- 提前写好的池子随机轮播，或特定日期指定内容

---

## 窗户区域

### 窗外风景

- 随真实时间变化（白天/黑夜）
- 随四季变化（春夏秋冬不同场景）

### 天气小青蛙

- 窗台上一只小青蛙
- 接入真实天气 API
- 晴天戴墨镜，雨天撑伞

### 小树苗

- 窗台上一盆植物
- 随果果年龄增长：种子→发芽→小苗→大树，每年自动"长大"

### 蒲公英

- 点击种子飘散（微交互）

### 窗帘

- 窗户开着时轻轻飘动，体现微风

### 萤火虫

- 夏天夜晚窗外飘动，偶尔飞进房间

## 四季系统

房间整体随真实季节变化：

| 季节 | 变化 |
|------|------|
| 春天 | 窗台花开、蝴蝶飞进来 |
| 夏天 | 桌上冰饮、风扇转动、猫摊成液体猫 |
| 秋天 | 窗外落叶、桌上多一个橘子 |
| 冬天 | 被子变厚、窗上霜花、猫缩成一团、热可可 |
